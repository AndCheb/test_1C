# Обзор проекта: Система фиксации деятельности персонала

## Название проекта
**Управление персоналом**

---

## Описание

1. Приложение содержит справочник "Персонал", который содержит ФИО сотрудника и его почасовую ставку.
2. По окончанию рабочего дня, сотрудник в документе "Отчет о работе" указывает время прибытия и время убытия, а так же вид проделанных работ из перечисления "Виды работ", содержащего следующие выды:
    + Обзвон заказчиков
    + Обзвон поставщиков
    + Контроль доставки товаров
3. Проведённые документы отражаются в периодическом (период 1 день) регистре сведений "Сведения о работе", в котором есть измерение "Сотрудник" и "Вид работ", а ресурсы "Количество часов" и "Сумма".
4. Сумма складывается из количества часов и почасовой ставке.
5. Необходим отчёт, который будет отражать список сотрудников, рабочие часы и суммы. Отчёт должен иметь отбор по периоду:
    + день
    + неделя
    + месяц
    + год

---
